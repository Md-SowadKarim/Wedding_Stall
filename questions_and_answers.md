###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

- C: 3
- D: 4

<p>

#### Answer: C

<i>here the foreach loop is running 4 times but the if condition is applying 3 times because the first value of nums is 0. and 0 means false, so the total value of count will be 3</i>
